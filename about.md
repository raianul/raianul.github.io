## Hi, I'm Raian

I'm a software engineer based in Berlin, Germany. I write about software engineering concepts — system design, object-oriented programming, data structures, databases, and networking — primarily in Bengali, because quality technical content in Bengali is still rare.

My goal is simple: make software engineering concepts accessible to Bengali-speaking learners worldwide, explained the way I wish someone had explained them to me.

## What's on this blog

- **OOP** — Object-oriented design principles and patterns
- **System Design** — Scalable architecture, trade-offs, real-world patterns
- **DSA** — Algorithms and data structures with intuition-first explanations
- **Databases** — Storage internals, query optimization, schema design
- **Networking** — Protocols, distributed systems fundamentals

Most posts have an English version too (toggle at the top of any article).
